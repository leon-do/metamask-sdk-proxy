## Setup

Install SDK

```
yarn add file:./metamask-sdk-v0.0.2.tgz
or
npm i file:./metamask-sdk-v0.0.2.tgz
```

Install Dependencies

```
yarn
or
npm i
```

Run proxy

```
node index.js
```
